# Banabells Task PWA

A premium mobile-first to-do list Progressive Web App built with HTML, CSS, Tailwind, vanilla JavaScript, and LocalStorage.

## Run locally

Open the project folder in VS Code and launch Live Server on index.html.

## Publish to GitHub Pages

1. Create a GitHub repository.
2. Push this folder to the repository.
3. In GitHub, go to Settings → Pages.
4. Choose the main branch and the root folder.
5. Your app will be available at:
   https://<your-username>.github.io/<repository-name>/

## Features

- Premium dark glass UI
- Task CRUD and filtering
- Calendar, statistics, and settings pages
- LocalStorage persistence
- PWA installability and offline support
